Graphical User Interface

The point cloud viewer consists of several elements:

  • at the center, main window to visualize point cloud data,
  • top-left, text box to search for a location
  • top-right, minimap
  • bottom-left, export points, settings, this help, and measure
  • bottom-middle, two buttons to switch between Google Earth-like and keyboard interaction modes (see below)

Navigation

 Google Earth-like controls

  • use left/right mouse buttons to move or rotate the camera
  • use the scroll wheel to zoom in/out

 Keyboard interaction

  • mouse right-click rotates camera
  • use arrow or a/s/d/w keys to move left, backward, right or forward, respectively
  • use q/z keys to move up or down
  • use mouse scroll wheel to increases or decreases the field of view
  • hold the spacebar to move faster

Export points

Select and export points in the LAZ file format. The download URL will be mailed to you as soon as the processing is completed. The selection of points can done in three ways:
  • by clicking on two points in the minimap,
  • by drawing a box in the 3D point cloud visualization; this only works in the Keyboard interaction navigation mode,
  • by entering the actual coordinates into the input form.
Note that the coordinates are based on the Amersfoort/RD new coordinate system. The number of points in the selected area can be equal to or less than the number indicated. The coverage indicates the percentage of point cloud data available for download. Since the number of points one can download is limited ([N]), the resolution of the point cloud data may differ depending on the size of selected area. Therefore, small areas are more likely to be obtained in full resolution (i.e. 10 points per m2) than large areas.

Settings

Settings are grouped into three tabs.

Point

Configure the "look and feel" of points.

Measurement

Configure how measurements are taken.

Misc(ellaneous)

Adjust other settings.

Search Location

Locate a landmark (e.g. city, address) by . You can also view the data for your current location (geolocation must be enabled in your browser).

Minimap

Shows the camera field of view as a red trapezoid. Aerial photos are shown by default. Change to road map using the layer switcher (top right corner). Click to enlarge the minimap.

Measurement

Toolkit
  • click: open measuring instruments
Distance
  • left mouse: insert a measurement point
  • right mouse: complete insertion
  • drag&drop: move measurement points
Surface area
  • left mouse: insert a measurement point
  • right mouse: complete insertion
  • drag&drop: move measurement points
The area is calculated on the ground plane. The height does not affect the result. Crossings should be avoided.
Volume
  • left mouse: insert volume or select inserted volume
  • switch between translation (icon), scale (icon) and rotation (icon) mode
  • keyboard shortcuts: T, E, R
The volume is shown on the object's centroid.
Height profile
  • left mouse: insert profile marker
  • right mouse: complete insertion
  • drag&drop: move profile marker
  • ctrl+drag&drop: drag upwards/downwards to increase/decrease the profile width
Depending on the _Clip mode_, points inside the profile will be highlighted whereas points outside the profile will be clipped.
Clip volume
  • left mouse: insert volume or select inserted volume
  • Switch between translation (icon), scale (icon) and rotation (icon) mode
  • keyboard shortcuts: T, E, R
