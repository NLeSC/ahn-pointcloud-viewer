Inspect Mode

When doubleclicking a site, the navigation options switch to 'inspect mode', this changes the mouse input to 'orbit' around the selected site and shows the information for this site.
Exit Inspect Mode
When inspecting a specific site, this button can be used to exit the inspect mode and return to the normal controls.

Search Box

In the search box, one can search all (textual) properties of the site objects, plus the site ID-numbers. There are several possibilities for searching. You can search for words or phrases, or use "tags" to search in specific fields. The supported tags are "site:", "condition:", "material:" and "time:". Below we list some example searches. Note that all searches are case-insensitive.

pyramid
Searches all textual fields of the site objects for the word "pyramid".
pyramidal monument
Searches all textual fields of the site objects for the exact phrase "pyramidal monument". Only objects containing this exact phrase will be found! For instance, if the description contains "pyramidal funerary monument", it will not be found by searching for "pyramidal monument".
site:21
Matched the site_id field of the site objects to the given number (21 in this case).
condition:complete
Searches the condition field of the site objects for the word "complete" (or "damaged").
material:tufo
Searches the material fields (material_type, material_subtype and material_technique) of the site objects for the word "tufo" or "peperino".
time:A.D
Searches the time fields (date_specific and period) of the site objects for the "A.D".
time:2nd time:century
When looking for multiple words using tags, one needs to add the tag to all words!
